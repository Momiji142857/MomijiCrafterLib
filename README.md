# MomijiCrafterLib

为 Mindustry Java 模组作者提供的多功能工厂方块库，可通过 JSON 配置快速使用。

## 现有方块

**MultiCrafter** —— 多配方切换工厂，一次启用一个配方，支持物品、液体、电力、热量、载荷的任意组合。自动适配原版单位建造速度与消耗规则。
**OmniCrafter** —— 单配方全能工厂，任意组合输入输出, 做了一半了
